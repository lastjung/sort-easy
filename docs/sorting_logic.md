# 정렬 알고리즘 시각화 통합 요구사항

## 1. 공통 시각화 규칙 (Consistency)

- **1-based 번호 체계**: 모든 막대 내부와 설명 메시지는 `1번, 2번...` 과 같은 위치 번호로 통일합니다.
- **색상 우선순위**: **빨강(교환/결정)** > **노랑(비교/탐색)** > **보라(기준점)** > **녹색(정렬완료)**.
- **바통 터치 연출**: 각 라운드 종료 시 다음 기준점을 미리 보여주는 `보라-보라` 전이 효과를 공통 적용합니다.
- **메시지 톤**: 정중한 한글 격식체로 통일하여 프리미엄 교육 도구의 느낌을 줍니다.

## 2. 알고리즘별 핵심 로직

### 버블 정렬 (Bubble Sort)

- **비교**: 인접한 두 막대를 노란색으로 비교합니다.
- **교환**: 앞이 더 크면 빨간색으로 교환하며 뒤로 밀어냅니다.
- **안착**: 라운드 종료 시 가장 큰 값이 끝에 보라색으로 마킹된 후 녹색으로 확정됩니다.

### 선택 정렬 (Selection Sort)

- **타겟**: 현재 채워야 할 자리를 보라색으로 마킹합니다.
- **탐색**: 나머지 구역을 노란색으로 훑으며 가장 작은 값을 찾습니다. 새로운 최소값은 빨간색으로 표시합니다.
- **교환**: 찾은 최소값과 타겟 자리를 빨간색으로 교환하고 녹색으로 확정합니다.

### 삽입 정렬 (Insertion Sort)

- **타겟**: 현재 정렬 구역에 삽입할 요소를 보라색으로 마킹합니다.
- **역방향 버블링**: 보라색 요소가 앞쪽의 녹색 구역을 노란색으로 비교하며 빨간색으로 교환하며 파고듭니다.
- **확정**: 제 자리를 찾으면 해당 구역 전체를 녹색으로 확정합니다.

## 3. 알고리즘 구호 (Slogans)

- **Bubble Sort**: "Spot the largest values bubbling to the top!"
- **Selection Sort**: "Hunting for the smallest number to put first!"
- **Insertion Sort**: "find small number by reverse bubbling sort."
